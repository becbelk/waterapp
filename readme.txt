هذه النسخة نسخة أولية مخصصة كتطبيق لمعالجة حسابات استهلاك المياه
للتثبيت يجب على المهتم 
أولا أن تكون له نسخة من
 nodejs  إصدار 18 أو أكثر
من اﻷفضل أن يتم تثبيت المكتبة
nodemon 
بعد التثبيت يجب التأكد من ربط التطبيق بالارتباطات =المكتبات= التالية
والتي يمكن تثبيتها جميعا عن طريق الامر

npm i dotenv express ejs  express-ejs-layouts
     express-session  
     cookie-parser jsonwebtoken 
     path blob-stream pdfkit pdfkit-table tafgeetjs 
     mongoose  connect-mongo method-override bootstrap brfs 
    في حالة عدم وجود مكتبة تثبت مثل ماسبق
    npm i [اسم المكتبة]
    تفعيل خدمة mongod في نظام التشغيل
    ولتشغيل البرنامج  اكتب
    node app.js
